Comandos
Comando creacion de ambiente de desarrollo
python -m venv env

Crear en environment
env\Scripts\activate

Instalar Django
pip install django

Instalar Django framework
pip install djangorestframework

Crear una carpeta donde va el proyecto
django-admin startproject nombreproyecto

Crear un modulo en django
django-admin startapp autorizacion

Instalar la libreria para SIMPLE jwt
pip install djangorestframework-simplejwt 

Instalar libreria de postgress
pip install psycopg2

Instalar requirement
pip install -r requirements.txt

Correr el servidor
python manage.py runserver

Crear migraciones que Django a DB
python manage.py makemigrations

Migrar lo que Django a DB
python manage.py migrate

Crear usuario administrador
python manage.py createsuperuser
